# vue-til-server
> spring + vue.js를 이용한 게시판

## 기능
+ 회원가입
+ 로그인
+ 게시글 작성, 조회, 수정, 삭제
+ 친구추가
+ 쪽지기능

## 개발 환경
+ Java(Jdk 11)
+ Spring boot, Spring security
+ JPA
+ Gradle
+ H2
+ Intellij 2022
